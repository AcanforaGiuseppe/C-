-English-
Visual Studio 2019's projected game:
Open the Visual Studio Solution file "GameJam.sln" and press "Execute" / "F5" for start the game,
or check Solution Explorer the Graphic Assets, sounds, and the code.

-Italian-
Gioco progettato con Visual Studio 2019:
Aprire il file Visual Studio Solution "GameJam.sln" e premere "Avvia" / "F5" per avviare il gioco,
oppure controllare in Esplora Soluzioni gli Assets grafici, i suoni, ed il codice.
